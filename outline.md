# 中医体质测试系统项目概述

## 项目目标
为www.dominical.cn网店开发一个基于中医体质养生理论的精准体质测试系统，可以嵌入到现有网站中使用。

## 核心功能
1. 科学的中医体质测试问卷（基于王琦教授九种体质分类法）
2. 精准的体质分析算法
3. 个性化的养生建议和产品推荐
4. 站内产品链接集成

## 九种体质类型
1. **平和质** - 健康平衡状态
2. **气虚质** - 元气不足，容易疲劳
3. **阳虚质** - 阳气不足，畏寒怕冷
4. **阴虚质** - 阴液亏少，口干咽燥
5. **痰湿质** - 痰湿凝聚，体型偏胖
6. **湿热质** - 湿热内蕴，面垢油腻
7. **血瘀质** - 血行不畅，面色晦暗
8. **气郁质** - 气机郁滞，情志不畅
9. **特禀质** - 先天失常，过敏体质

## 技术架构
- 前端：HTML5 + CSS3 + JavaScript
- 样式：Tailwind CSS
- 动画：Anime.js
- 可视化：ECharts.js
- 响应式设计，支持移动端

## 页面结构
1. **index.html** - 测试主页面
2. **result.html** - 测试结果页面
3. **products.html** - 产品推荐页面

## 特色功能
- 渐进式测试体验
- 实时进度显示
- 可视化体质分析图表
- 个性化产品推荐
- 站内链接无缝集成
